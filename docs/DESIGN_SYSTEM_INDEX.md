# Chatwoot Design System Documentation

Welcome to the Chatwoot Design System documentation! This comprehensive guide is designed to help developers, designers, and AI agents build pixel-perfect UI components that match Chatwoot's design language.

## 📚 Documentation Files

### Complete Design System Guide
**[DESIGN_SYSTEM.md](../DESIGN_SYSTEM.md)** - Comprehensive design system documentation

Covers:
- Design principles and visual style
- Complete design token reference (colors, spacing, typography)  
- Layout system and grid patterns
- Full component library specifications
- Accessibility guidelines (WCAG 2.1 AA)
- Dark mode implementation
- Animation specifications
- Framework-agnostic implementation notes

**Best for:** Understanding the complete system, implementing from scratch, design decisions

### AI Agent Quick Reference  
**[AI_DESIGN_GUIDE.md](../AI_DESIGN_GUIDE.md)** - Fast lookup for LLM-powered development

Includes:
- Copy-paste ready component templates
- Quick color, size, and spacing reference
- Common pattern snippets
- Icon quick reference
- State pattern examples
- Implementation checklists

**Best for:** Rapid development, AI-assisted coding, quick prototyping

## 🚀 Quick Start

1. **First-time users:** Read [DESIGN_SYSTEM.md](../DESIGN_SYSTEM.md) sections 1-4
2. **AI agents:** Start with [AI_DESIGN_GUIDE.md](../AI_DESIGN_GUIDE.md) System Summary
3. **Quick lookup:** Bookmark [AI_DESIGN_GUIDE.md](../AI_DESIGN_GUIDE.md) for templates

## 🎯 Key Information

- **Framework:** Vue 3 Composition API + Tailwind CSS
- **Components:** 47+ in `app/javascript/dashboard/components-next/`
- **Color System:** Radix UI semantic tokens (12-step scales)
- **Icons:** Iconify/Lucide  
- **Dark Mode:** Automatic with semantic tokens

## 📖 Common Tasks

**Create a button:** Copy template from AI_DESIGN_GUIDE.md → Button  
**Build a form:** Use AI_DESIGN_GUIDE.md → Form Layout  
**Understand colors:** Read DESIGN_SYSTEM.md → Color System  
**View components:** Run `pnpm story:dev`
